# Mad-Libs

Description
Mad Libs is a C# console application that lets you create hilarious stories by prompting you to provide various words (nouns, adjectives, verbs, etc.). The program then uses your inputs to fill in the blanks in a predefined story template, resulting in a funny and personalized story.

Features
Interactive command-line interface for user input.
Customizable story template with placeholders for user inputs.
Multiple prompts for different word types (adjectives, nouns, verbs, etc.).
Entertaining and creative stories generated based on user inputs.

Installation
Clone the repository to your local machine:
git clone https://github.com/BernhardTuft/mad-libs.git
Navigate to the project directory:
cd mad-libs
Build the project:
dotnet build

Usage
Run the program: dotnet run
Follow the prompts to enter various words (adjectives, nouns, verbs, etc.) as requested by the program.
Once all prompts are completed, the program will generate the Mad Libs story using your inputs.
Enjoy reading your hilarious and personalized story!
